# Software Engineering Project

A web-based business website developed using PHP, HTML, CSS, JavaScript, and MySQL. The project provides information about the company, its services, products, and allows visitors to contact the business through an online contact form.

## Features

- Home page
- About Us page
- Services page
- Products page
- Contact Us page
- Contact form processing with PHP
- Database connectivity using MySQL
- Responsive user interface
- Organized project structure with separate CSS, JavaScript, and resource folders

## Installation

### Prerequisites

- XAMPP installed
- Apache and MySQL running
- Web browser

### Setup Steps

1. Clone the repository:

```bash
git clone https://github.com/ydiallo0821/Software-engineering-Project.git
```

2. Move the project folder into the XAMPP htdocs directory:

```text
C:\xampp\htdocs\
```

3. Start Apache and MySQL from XAMPP Control Panel.

4. Create the required database in phpMyAdmin.

5. Update database credentials in:

```php
database.php
```

6. Open the project in your browser:

```text
http://localhost/software-eng-proj-1/final.php
```
